# Credit Card Comparison Project

A Streamlit-based web application to compare credit cards, visualize benefits, and find the best card for your needs.

## Quick Start

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the App**:
    ```bash
    streamlit run streamlit_app/Home.py
    ```
    *(Adjust the filename if your entry point is different, e.g., `main.py` or `Hello.py`)*

## Features
- **Card Comparison**: Side-by-side comparison of fees, salaries, and benefits.
- **Visuals**: Glassmorphism UI design for a modern look.
- **Search**: Filter cards by bank, salary, or type.

## Structure
- `streamlit_app/`: Source code for the frontend.
- `data/`: Database or CSV files containing card information.
